**Вариант 9**  
**Фамилия: Семешко**

---

## Общая информация

Лабораторная работа выполнена в одном проекте.  
Все задания реализованы в виде отдельных классов, сгруппированных по логическим пакетам.  

В программе присутствует **единственная точка входа** — класс `Main4Var9`.  
В главном методе `main` продемонстрирована работа всех классов и методов  
с использованием **дружественного интерфейса**.

Все исходные данные вводятся **с клавиатуры**, для каждого ввода реализована  
**проверка корректности данных**.

В каждом классе присутствуют:
- свойства (поля);
- конструкторы;
- метод `toString()`.

## Задание 1.9  
**Диапазон оценок**

### Условие  
Изменить сущность **Студент** таким образом, чтобы ему можно было  
добавлять только оценки в диапазоне от **2 до 5**.  
При этом всегда должна быть возможность получить список оценок.  
Необходимо гарантировать, что некорректную оценку задать невозможно.

### Реализация  
- Класс `Student` (`ru.semeshko.students`).
- Поле с оценками объявлено как `private`.
- Добавление оценки выполняется только через метод `addGrade(int)`.
- В методе реализована проверка диапазона `[2;5]`.
- Метод `getGrades()` возвращает копию массива.

### Демонстрация  
Ввод оценок осуществляется с клавиатуры.  
Попытка добавить некорректную оценку отклоняется.

### Вывод  
Некорректные оценки добавить невозможно, состояние объекта защищено.

## Задание 2.3  
**Жёлтые страницы (Телефонный справочник)**

### Условие  
Разработать сущность для хранения пар  
`телефон – имя` и поиска телефона по имени.  
Если имя уже существует, телефон заменяется, старый телефон возвращается.

### Реализация  
- Класс `PhoneBook` (`ru.semeshko.phonebook`).
- Используется `Map<String, String>`.
- Реализованы операции:
  - добавление и замена контакта;
  - удаление контакта;
  - получение телефона по имени;
  - проверка наличия имени и телефона;
  - получение количества контактов;
  - получение массивов имён, телефонов и пар;
  - поиск имён по префиксу.

### Демонстрация  
Контакты вводятся с клавиатуры, выполняется поиск и вывод справочника.

### Вывод  
Телефонный справочник работает корректно.
